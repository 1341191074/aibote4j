## 🎯 API 分类

### 按平台分类
- **Windows API** - 针对 Windows 桌面应用的自动化操作
- **Android API** - 针对 Android 设备的自动化操作
- **Web API** - 针对 Web 应用的自动化操作

### 按功能分类
- **基础操作 API** - 连接、断开、休眠等基本操作
- **界面操作 API** - 点击、输入、滑动等界面交互
- **识别 API** - 图像识别、OCR、颜色查找等识别功能
- **系统 API** - 文件操作、系统控制等底层功能
- **工具 API** - 配置管理、缓存、追踪等辅助功能
